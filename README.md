<h1>Spyre Programming Langage</h1>
Spyre is a hobbist programming language.

<h3>Key features</h3>
<ul>
  <li>Statically typed</li>
  <li>Interpreted</li>
  <li>Garbage collected</li>
  <li>Procedural</li>
</ul>

<h3>Installation</h3>
<pre><code>git clone https://DavidWilson4242/spyre.git
cd spyre
make spyre
</code></pre>
